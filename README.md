# rag-knowledge-base
企业级 AI 知识库管理系统，基于 RAG 技术构建，支持智能文档处理、向量化存储和自然语言问答。
核心功能        
  - 文档管理：支持上传多种格式文档，自动解析、切片、向量化
  - RAG 智能问答：基于自有文档内容，通过 LLM 生成带来源引用的回答
  - 多租户架构：通过组织标签（OrgTag）实现数据隔离
  - 混合搜索：Elasticsearch 关键词搜索 + 向量语义搜索
  - 实时对话：基于 WebSocket 的流式对话
  - 异步处理：Kafka 消息队列驱动文档解析与向量化流水线
